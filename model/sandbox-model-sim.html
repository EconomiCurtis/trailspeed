Toblers hiking function

Walking velocity: \[ W=6e^{\displaystyle-3.5\left\vert\frac{dh}{dx}+0.05\right\vert} \]

\[ \frac{dh}{dx}=S=\tan\Theta \]

where

  • \(dh\) = elevation difference,
  • \(dx\) = distance,
  • \(S\) = slope,
  • \(\Theta\) = angle of slope (inclination).

Modified Toblers hiking function

Walking velocity: \[ V = V_m e^{\displaystyle-3.5\left\vert\frac{dh}{dx}+0.05\right\vert} \]

\[ \frac{dh}{dx} = S = \tan\Theta \]

where

  • \(dh\) = elevation difference,
  • \(dx\) = distance,
  • \(S\) = slope,
  • \(\Theta\) = angle of slope (inclination).
  • \(\phi\) = fatigue.
  • \(V_m\) = maximum speed.

A slope beyond 90 degrees means you are either rock climbing or falling.

walkspeed <- function(slope = -0.05, 
                      V_m = 3.4, # max speed
                      bSlope = 0.05, # fastest slope
                      phi = 1.5, # fitness
                      week = 1 # of hiking
                      ){
  (V_m - phi/week) * exp((-3.5) * abs(tan(slope) + bSlope))
}

sim = data.frame(
  slope = c(
    runif(5000, -0.8,0.8),
    rnorm(6000, 0,0.2)
  ))
sim$week = runif(length(sim$slope), 1, 10)

  
sim <- sim %>%
  mutate(
    speed = walkspeed(
      slope = slope,
      V_m = 3.4, 
      bSlope = 0.05,
      #phi = 0.99,
      week = week
    ) + rnorm(n(), 0, sd = 0.05)
  ) %>%
  tbl_df()


ggplot(
  data = sim,
  aes(
    x = slope,
    y = speed,
    colour = week
  )
) + 
  geom_point() +
  theme_minimal() +
  xlab("Slope (in degrees)") +
  ylab("Walking Speed (mph)")

reg <- lm(
  log(speed) ~ tan(abs(slope)),
  data = sim
)
